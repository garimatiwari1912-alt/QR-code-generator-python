# QR Code Generator (Python)

A simple Python project that generates QR codes from text or links.

## 📌 Features

* Generate QR codes from any text or URL
* Save QR code as image file
* Lightweight and easy to use

## 🛠️ Requirements

Install required library before running:

```bash
pip install qrcode[pil]
```

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/QR-code-generator-python.git
```

2. Go to project folder:

```bash
cd QR-code-generator-python
```

3. Run the script:

```bash
python qr_gen.py
```

## 📷 Output

The program will generate a QR code image file in the same folder.
